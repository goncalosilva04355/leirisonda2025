🔥 Como Habilitar Firestore

Projeto: leiria-1cfc9

📋 Passo 1 🔥 Passo 2 ⚙️ Passo 3 ✅ Passo 4 🧪 Verificar

⚠️ Problema Atual:

Erro: "Service firestore is not available"

Causa: O Firestore não está habilitado no projeto Firebase

Solução: Seguir os passos abaixo para habilitar

📋 Passo 1: Acessar Firebase Console

Clique no link abaixo para ir diretamente ao seu projeto Firebase:


🔗 Abrir Firebase Console

URL completa:

https://console.firebase.google.com/project/leiria-1cfc9

💡 Certifique-se de estar logado com a conta Google correta

🔥 Passo 2: Navegar para Firestore

No Firebase Console, no menu lateral esquerdo:

  1. Procure a seção "Criação"
  2. Clique em "Firestore Database"

📱 Menu lateral:

Criação → Firestore Database

💡 Se não vir "Firestore Database", role o menu para baixo

⚙️ Passo 3: Criar Base de Dados

Verá uma página com o botão "Criar base de dados"

3.1. Clique em "Criar base de dados"

3.2. Escolher Modo de Segurança:

  • Modo de teste: ✅ Recomendado para desenvolvimento
  • Modo de produção: ❌ Mais seguro mas requer configuração
💡 Recomendação: Escolha "Modo de teste" para facilitar o desenvolvimento. As regras podem ser alteradas depois.

3.3. Escolher Localização:

Localizações recomendadas para Portugal:

  • europe-west3 (Frankfurt) - ✅ Recomendado
  • europe-west1 (Bélgica) - ✅ Alternativa
  • europe-west2 (Londres) - ✅ Alternativa

✅ Passo 4: Confirmar Criação

  1. Revise as configurações selecionadas
  2. Clique em "Concluído" ou "Done"
  3. Aguarde a criação da base de dados (pode demorar 1-2 minutos)
🎉 Sucesso! Quando vir a interface do Firestore com coleções vazias, o processo está completo.

🧪 Verificar se Funcionou

Após habilitar o Firestore:

  1. Recarregue esta aplicação (F5 ou Ctrl+R)
  2. Os erros "Service firestore is not available" devem desaparecer
  3. Os dados começarão a ser salvos no Firestore automaticamente
🏠 Voltar à App 🧪 Testar Firestore 🔧 Debug Firestore

🔍 Resolução de Problemas

❓ Não vejo "Firestore Database" no menu

  • Certifique-se de estar no projeto correto: leiria-1cfc9
  • Role o menu lateral para baixo
  • Procure na seção "Criação" ou "Build"

❓ Erro de permissões

  • Verifique se está logado com a conta correta do Google
  • Certifique-se de ter permissões de administrador no projeto
  • Contacte o proprietário do projeto se necessário

❓ Firestore criado mas app ainda não funciona

  • Aguarde 2-3 minutos para propagação das alterações
  • Recarregue completamente a aplicação (Ctrl+F5)
  • Limpe o cache do browser se necessário
